# Jarvis Voice Assistant (Simple)

This is a basic voice assistant made in Python. It listens to your voice and performs simple tasks like playing songs, telling time/date, and answering questions.

---

## Features

* Play songs on YouTube
* Tell current time and date
* Answer "who is" questions using Wikipedia
* Simple conversation

---

## Installation

Install required libraries:

```bash
pip install pyttsx3 pyaudio SpeechRecognition pywhatkit wikipedia
```

---

## Run the Program

```bash
python jarvis_assistance.py
```

---

## How to Use

1. Run the program
2. Say **"Jarvis"** to activate
3. Give a command like:

   * "Jarvis play song"
   * "Jarvis what is time"
   * "Jarvis who is Elon Musk"

---

## Note

* Internet is required
* Microphone must be working
* Voice recognition may not always be accurate
